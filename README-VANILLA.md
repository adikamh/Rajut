# Toko Rajut - Vanilla HTML Version (Mobile Optimized)

Versi sederhana dari website Toko Rajut yang menggunakan HTML, CSS, dan JavaScript murni dengan optimasi penuh untuk perangkat mobile.

## 🚀 Fitur Mobile

### Touch-Friendly Design
- **Touch targets minimum 44px** - Semua button dan link mudah diklik dengan jari
- **Responsive typography** - Font size menyesuaikan layar
- **Mobile-first approach** - Design dibuat untuk mobile terlebih dahulu

### Navigation Mobile
- **Hamburger menu animasi** - Menu burger dengan animasi smooth
- **Slide-down menu** - Menu muncul dengan efek slide dari atas
- **Auto-close menu** - Menu tertutup otomatis saat klik di luar

### Gesture Support
- **Swipe navigation** - Geser kiri/kanan untuk navigasi antar halaman
- **Pull-to-refresh** - Tarik ke bawah untuk refresh halaman
- **Touch-optimized lightbox** - Gallery dengan lightbox yang responsive

### Mobile Optimizations
- **Viewport height fix** - Mengatasi masalah tinggi viewport di mobile browser
- **Passive event listeners** - Scroll performance yang lebih baik
- **Orientation change support** - Menyesuaikan saat rotasi layar
- **PWA-ready meta tags** - Siap untuk Progressive Web App

## File Structure

- `index-vanilla.html` - File HTML utama
- `style.css` - Styling CSS custom
- `script.js` - JavaScript untuk interaktivitas

## Perbedaan dengan Versi React

Versi ini adalah konversi sederhana dari aplikasi React asli dengan fitur-fitur berikut:

- ✅ Navigasi antar halaman
- ✅ Layout responsif
- ✅ Form contact
- ✅ Gallery dengan gambar
- ❌ Authentication (login/logout)
- ❌ Dynamic data dari PocketBase
- ❌ Admin dashboard
- ❌ Routing kompleks

Untuk fitur lengkap, gunakan versi React asli dengan `npm run dev`.
