## 🚀 Quick Deploy ke GitHub Pages

### Langkah Cepat:
1. **Fork** atau **Clone** repository ini
2. **Upload** ke GitHub repository Anda
3. **Aktifkan GitHub Pages** di Settings → Pages
4. **Akses** website di `https://[username].github.io/[repo-name]/`

### Detail Setup:
```bash
# Clone repository
git clone https://github.com/adikamh/Rajut.git
cd Rajut

# Buat repository baru di GitHub
# Upload semua file ke repository Anda

# Aktifkan GitHub Pages di Settings
```

## 🔧 Customization

### Mengubah Konten:
1. Edit `index.html` untuk text dan struktur
2. Update `style.css` untuk colors dan styling
3. Modify `script.js` untuk functionality

### Menggunakan Custom Domain:
1. Buat file `CNAME` dengan domain Anda
2. Upload ke repository root
3. Configure DNS di domain provider

### Mengganti Gambar:
- Ganti URL Unsplash di `index.html`
- Atau upload gambar ke repository dan update path
